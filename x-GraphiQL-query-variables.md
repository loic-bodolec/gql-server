### Get single course

```javascript
{
    "courseID" : 1
}
```

### Get courses by title

```javascript
{
  "title": "JavaScript: Understanding The Weird Parts"
}
```

### Get courses (with fragments)

```javascript
{
    "courseID1": 1,
    "courseID2": 2
}
```

### Add course

```javascript
{
  "id": 4,
  "title": "hello",
  "author": "loicbdev",
  "description": "text-test",
  "topic": "Vue",
  "url": "https://vuejs.org/"
}
```

### Update course topic

```javascript
{
  "id": 1,
  "topic": "JavaScript"
}
```
